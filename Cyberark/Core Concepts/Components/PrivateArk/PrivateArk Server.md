The PrivateArk Server and the PrivateArk Client are two of the core components of the CyberArk Privileged Access Security solution. The PrivateArk Server is the central repository for all privileged credentials, while the PrivateArk Client is the interface that users use to access and manage those credentials.

- Server Central Administration control
- The red and green light indicate the vault is active or inactive
- The security algorithm used is SHA2-256/SHA1
